# 视觉设计系统

## 设计原则
- 视觉为叙事服务
- 一页一个核心观点
- 信息密度适中：字少、图大、留白足

## 页面类型

| 类型 | 用途 | 结构 |
|---|---|---|
| Hero | 开场/章节转场 | 大标题 + 一句话 + 大背景色块 |
| Point | 阐述一个观点 | 标题 + 3 个以内要点 + 可选图标 |
| Chart | 展示数据 | 标题 + 图表 + 一句洞察 |
| Quote | 强调金句 | 引用文字 + 出处 |
| Closing | 结尾 | 核心信息 + 行动号召 |

## 配色方案
根据风格选择一套：
- **专业沉稳**：深蓝 `#1a365d` + 浅灰 `#f7fafc` + 强调色 `#c53030`
- **科技现代**：深黑 `#0d1117` + 青绿 `#00d9ff` + 白色
- **温暖亲和**：米白 `#faf9f6` + 砖红 `#c05621` + 深蓝 `#2c5282`

## 字体
- 中文标题：Microsoft YaHei / PingFang SC / Noto Sans SC
- 中文正文：Microsoft YaHei / Source Han Sans
- 英文标题：Montserrat / Inter / Helvetica
- 英文正文：Inter / Segoe UI

## 一致性检查
- 全文字体不超过 2 种
- 每页主色不超过 3 种
- 标题层级统一
- 页边距一致
